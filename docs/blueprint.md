# **App Name**: TaskFlow

## Core Features:

- Add New Task: Users can easily input and add new tasks to their to-do list.
- View Task List: Display all active and completed tasks in a clear, sortable list.
- Mark Task Complete: Users can toggle the completion status of a task, moving it between active and completed states.
- Delete Task: Provide an option to remove unwanted tasks from the list.
- Task Data Storage: Persist tasks and their states locally, so they are available across user sessions.
- AI Task Categorization: An AI tool that suggests relevant categories for new tasks based on their descriptive input.

## Style Guidelines:

- The primary color, conveying focus and clarity, is a vibrant but calming blue: #226BBF (HSL: 220, 70%, 45%).
- A light and airy background color provides a clean canvas, derived from the primary hue for cohesion: #ECF1F6 (HSL: 220, 15%, 96%).
- An accent color, distinct yet harmonious, is a bright cyan for interactive elements and highlights: #16BBDB (HSL: 190, 80%, 55%).
- Headline and body font: 'Inter' (sans-serif) for its modern, clear, and objective readability, promoting an efficient user experience.
